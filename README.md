# 序言

1. 你觉得自己曾经遇到的BUG再次出现时你能顺利解决的可能性有多大？
2. 你觉得自己的记忆力可以记住各种细节问题吗？
3. 您自己有个人博客记录这些曾经遇到的BUG吗？
4. 你觉得自己有时间记录自己解决问题的过程吗？
5. 如果你有机会参与个人开发问题的整理分享你会参与吗？
